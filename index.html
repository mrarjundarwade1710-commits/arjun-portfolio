Learning Approach

My learning approach focuses on understanding concepts from their fundamental principles. Whenever I learn a new topic, I begin with the mathematical intuition behind the concept, then understand the conceptual or geometric interpretation, and finally implement the concept from scratch using programming.

For example, when studying algorithms or statistical techniques, I first study the mathematical formulation, understand how the concept behaves conceptually, and then build the implementation step by step in Python before using advanced libraries.

This approach helps develop deeper understanding of computer science, statistics, and data analysis rather than relying only on pre-built tools.
